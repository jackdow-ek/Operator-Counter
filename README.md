# Operator-Counter
Java uzantili  dosyadaki sayisal, mantiksal ve ilişkisel operatörlerin sayisini hesaplayan console uygulaması. Hesaplamanın yapılacaği dosya  console üzerinden komut satırı olarak girilmelidir.
